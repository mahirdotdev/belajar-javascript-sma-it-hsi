# Variable Dalam Javascript

## ATURAN PENULISAN VARIABLE

1. Hanya Boleh berisi huruf dan angka
2. tidak boleh menggunakan tanda baca kecuali _ (underscore) dan $ (dollar sign)
3. case sensitif (peka dengan huruf besar dan kecil contoh A tidak sama dengan a)
4. bisa dengan pola
    - camelCase
    - PascalCase
    - snake_case / SNAKE_CASE
5. diawali dengan let/const/var + spasi + nama_variable + tanda sama dengan + value
    const nama="Budi"

Contoh Salah
x. andi&budi =
x. andi budi =
